# covid_prevention_management_system
# 防疫需求信息管理系统

我们致力于构建一个名为疫知通的疫情防控管理系统，面向普通用户、疫情防控单位、管理员，使疫情防控管理、物资管理和核酸检测管理变得更加便捷。系统主要提供的功能为以下几点：用户信息管理、物资捐赠、个人物资需求、物资管理、医疗管理和志愿管理等。
